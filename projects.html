Bloggy

This is a social networking app created using the Flask web framework in Python. Users can sign up and create blog posts, update the posts, and delete them. Users have the ability to create a bio about themselves for others to click on. This web application uses Python as the backend and HTML, CSS, and Bootstrap as the frontend. It is using a SQLite database. The GitHub repository is here!
The live demo is here!

Todo-List-App

This is a Todo-List-App created using the Flask web framework in Python. Users can sign up, create todos, and mark them complete once they have been finished. This app is able to perform CRUD operations. This web application uses Python as the backend and HTML, CSS, and Bootstrap as the frontend. It is using a SQLite database. Air Quality Monitor
